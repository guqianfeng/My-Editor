# My Editor

# 目录结构

* index.html - 模板页面

* src
    * command
        * index.ts - 简单封装document.execCommand
    * config
        * menu - 菜单配置
        * text - 编辑文本区域配置
        * index - 汇总上述配置
    * editor
        * index.ts - 核心类
        * index.scss - 样式
